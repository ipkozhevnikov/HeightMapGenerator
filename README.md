# Генерация карты высот

## 📌 Описание
Система реконструкции поверхности для мобильного робота на основе данных с **4 стереокамер**.  
Формирует **карту высот (height map)** для навигации и анализа окружающей среды.

---

## ⚙️ Алгоритм работы
1. Загрузка карт глубины (depth map) с 4 камер  
2. Предобработка данных (фильтрация шума, игнорирование пустых значений)  
3. Преобразование пикселей в 3D-точки в системе координат камеры  
4. Перевод точек в глобальную систему координат робота  
5. Объединение (слияние) облаков точек со всех камер  
6. Проекция точек на плоскость (формирование сетки высот)  
7. Заполнение пропусков (интерполяция)  
8. Нормализация высот и цветовая визуализация  
9. Сохранение карты высот в PNG  

---

## 📊 Данные
**Вход:** 4 карты глубины (их можно загрузить из папки **пример_входных_изображений**) + параметры камер  
**Выход:** карта высот 1000×1000 (область 2×2 м, PNG)
